# fastro-cli

clone `fastro-web` template, install dependencies & build all typescript files

## Install
```
npm i fastro-cli -g
```

## Usage
Clone template, create .env, install deps & build all typescript files:
```
fastro init
```
Get fastro-cli version:
```
fastro --version
```
